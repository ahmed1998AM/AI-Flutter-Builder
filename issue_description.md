تطبيق إسلامي ذكي متكامل باسم "فقيه AI" (Faqih AI)

## وصف التطبيق
منصة إسلامية متكاملة تعمل بدون تسجيل دخول وتحتوي على:
- القرآن الكريم كامل مع القراء والصوت
- مواقيت الصلاة والأذان مع GPS
- الأذكار الصباحية والمسائية
- التسبيح الإلكتروني
- الموسوعة الإسلامية
- الذكاء الاصطناعي للأسئلة الإسلامية
- تفسير الأحلام
- البوصلة القبلية

## الشاشات المطلوبة
1. شاشة البداية (Splash Screen) - بشعار أنيميشن
2. الشاشة الرئيسية - dashboard يظهر مواقيت الصلاة، دعاء اليوم، آية عشوائية
3. شاشة القرآن الكريم - عرض السور + القراءة + البحث
4. شاشة مشغل الصوت - تشغيل القرآن مع تحكم بالخلفية
5. شاشة الأذان - مواقيت الصلاة مع GPS + إشعارات
6. شاشة الأذكار - أذكار الصباح والمساء مع عداد
7. شاشة التسبيح - عداد ذكي مع اهتزاز
8. شاشة الدعاء - أدعية متنوعة مع تصنيفات
9. شاشة الموسوعة الإسلامية - قصص الأنبياء، الصحابة، الحديث
10. شاشة "الفقيه AI" - محادثة ذكية إسلامية
11. شاشة مفسر الأحلام - تفسير الأحلام بالذكاء الاصطناعي
12. شاشة الإعدادات - ثيمات، لغة، API keys

## المكتبات المطلوبة
http, shared_preferences, provider, audioplayers, flutter_local_notifications, geolocator, flutter_qiblah, sqflite, google_generative_ai, path_provider, connectivity_plus, intl, url_launcher, lottie, flutter_background_service
